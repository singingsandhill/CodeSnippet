### DNS 미등록시 수기 Host 등록
1) hosts 파일 위치
   - 경로: C:\Windows\System32\drivers\etc\hosts

2) 관리자 권한으로 메모장 열기
   - 시작 메뉴에서 메모장 검색
   - 메모장에 우클릭 → 관리자 권한으로 실행

3) hosts 파일 열기
   - 메모장 메뉴 파일 → 열기
   - 경로에 C:\Windows\System32\drivers\etc\ 입력
   - 오른쪽 아래 파일 형식을 텍스트 문서(*.txt) → 모든 파일(.) 로 변경
   - hosts 파일 선택 후 열기

4) 맨 아래에 매핑 추가
   - <IP주소> <도메인>
