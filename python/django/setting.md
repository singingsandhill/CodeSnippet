### 가상환경 생성: 
```
python3.13 -m venv .hands_on----
```

### 가상환경 적용
```
source .hands_on/bin/activate
.hands_on\Scripts\activate
.\ch5\.hands_on\bin\Activate.ps1 
```

### Django 설치: 
```
pip install django
```

### 루트 디렉토리 생성: 
```
mkdir hands_on
```

### Django 프로젝트 생성: 
```
django-admin startproject config hands_on
```

### 프로젝트로 이동: 
```
cd hands_on
```

### 앱 생성: 
```
python manage.py startapp orm
```
